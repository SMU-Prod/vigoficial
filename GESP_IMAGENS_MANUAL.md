## PADRÕES DE UI IDENTIFICADOS (Crítico para Automação VIGI)

### Barra Universal de Ações (presente em TODOS os processos):
```
← | EXCLUIR | VERIFICAR | ENVIAR
```

### Diálogo de Confirmação Padrão:
```
"Confirma [ação]?" → Sim / Não
```

### Mensagem de Sucesso Padrão:
```
"Processo Enviado com Sucesso. Nº Processo: AAAA/NNNN"
```

### Estrutura de Formulário de Processo:
1. Painel lateral com abas (Dados Básicos, Guias de Recolhimento, [específicos], Documentos)
2. Formulário principal à direita
3. Botão "Salvar" em cada aba
4. Barra de ações no topo
5. Fluxo: Preencher → Salvar cada aba → VERIFICAR → ENVIAR → Confirmar

### Fontes de Arrecadação GRU (Códigos):
- 140244, 140252, 140260, 140279, 140295, 140309, 140325, **140368** (Vistoria Inst. Financeira)

### Limites de Arquivo:
- PDF: 1.5MB máximo
- Imagem (JPG): 150-250KB máximo

### Formato de Processo:
- Número: AAAA/NNNN (Ano/Sequencial)
